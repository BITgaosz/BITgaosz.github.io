## Selected Projects

### Rare-cell-aware Coreset for Single-cell Clustering

Designed a rare-cell-aware coreset framework for large-scale scRNA-seq clustering. The method combines graph connectivity, inverse-density weighting, and micro-cluster rebalancing to preserve rare cells, boundary cells, and transitional states under aggressive data reduction.

### Survey of Cell Type Annotation Methods

Systematically surveyed automatic cell type annotation methods for scRNA-seq data, covering reference-based, marker-based, clustering-based, traditional machine learning, DNN/GNN, Transformer, and LLM-based methods. Organized more than 70 representative methods into a structured comparison table.

### ChatGLM-based Chinese Function Word Usage Prediction

Built annotation guidelines and datasets for Chinese preposition and conjunction usage prediction, then converted the task into an instruction-tuning problem with ChatGLM-6B and P-Tuning. The model achieved 93.91% accuracy for preposition usage prediction and 94.71% for conjunction usage prediction on the test set.
