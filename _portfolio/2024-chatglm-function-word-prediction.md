This undergraduate thesis project studies automatic prediction of Chinese preposition and conjunction usage. It combines linguistic annotation guidelines with large language model instruction tuning.

## Contributions

- Built a function classification system and annotation guideline for Chinese prepositions and conjunctions.
- Annotated a dataset covering 109 prepositions and 93 conjunctions, with 10,468 preposition instances and 7,051 conjunction instances.
- Converted the usage prediction task into an instruction-following classification problem.
- Fine-tuned ChatGLM-6B using P-Tuning for parameter-efficient adaptation.
- Achieved 93.91% accuracy for preposition usage prediction and 94.71% accuracy for conjunction usage prediction on a 1,000-example test set.
- Analyzed errors caused by polysemy, incomplete context, and ambiguous functional boundaries.

## Keywords

ChatGLM-6B, P-Tuning, instruction tuning, Chinese NLP, function word usage prediction, dataset annotation.
