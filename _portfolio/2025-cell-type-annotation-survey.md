This project surveys automatic cell type annotation methods for single-cell RNA sequencing data. The goal is to understand the technical landscape and support method selection for AI4S-oriented single-cell analysis workflows.

## Scope

- Reviewed survey papers, method papers, tool documentation, and database resources for cell type annotation.
- Organized methods into reference-based, marker-based, clustering-based, traditional machine learning, DNN/GNN, Transformer, and LLM-based routes.
- Built a structured comparison table covering more than 70 representative methods, including CHETAH, SingleR, scCATCH, Garnett, scAnnotatR, scNym, scDeepSort, scPML, CellPLM, scAgent, CellReasoner, and CellTypeAgent.
- Compared methods by input dependency, database source, annotation granularity, model framework, scalability, interpretability, and ability to handle novel or rare cell types.

## Keywords

Cell type annotation, scRNA-seq, AI4S, GNN, Transformer, LLM Agent, bioinformatics survey.
